Map App
Map App is a simple web application that uses Leaflet.js and the OpenStreetMap Nominatim API to display an interactive map and allow users to search for locations. It dynamically updates the map view based on the entered location and displays a marker at the specified location.

Features
Interactive Map: Utilizes Leaflet.js library to display an interactive map.
Location Search: Allows users to enter a location (e.g., city name) and search for it on the map.
Marker Display: Places a marker on the map to indicate the searched location.
Dynamic Map View: Updates the map view to center on the searched location with appropriate zoom level.
