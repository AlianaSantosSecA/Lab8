
Country Flag Quiz is a web application that quizzes users on their knowledge of various country flags. It displays flag images and multiple-choice options for each question, allowing users to select their answers. It provides immediate feedback on whether the chosen answer is correct or incorrect, keeps track of the score, and displays the overall score upon completing all questions.

Features
Interactive Quiz: Presents multiple-choice questions about country flags.
Flag Display: Shows the flag image associated with each question.
Feedback: Provides immediate feedback on the correctness of the chosen answer.
Score Tracking: Tracks the number of correct answers and displays the overall score at the end.
Responsive Design: Designed to be responsive and work well across different devices.
